Tempus Rattus

Rattus vita. Rattus sickus. Rattus mortis.

Description

Tempus Rattus is a game set in 1369 England, where you play as a plague infested rat on a mission to spread the black death to unsuspecting townsfolk. Unfortunately for you, the local plague doctor has other plans. I'd move quickly if I was you.

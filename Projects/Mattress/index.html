Mattress Physics Engine

A C++ based physics engine built from scratch for Bed Engine

Description

Mattress is a custom-built physics engine developed for Bed Engine. It is based around C++ and the aim is to make physics as trivial as possible for the user.

Technologies Used

C++ Custom Math Lib

Features

  • Euler implicit integration

Future Developments

  • Euclidean and Non-Euclidean Physics
  • Gravity manipulation
